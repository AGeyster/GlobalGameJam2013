Global Game Jam 2013 - Arrythmia Ninja

Project created using C# in Microsoft Visual Studio 2010 Express

Project used Microsoft's XNA 4.0 game library and platformer starter kit, code and content copyright to Microsoft

Additional Code done by myself, Andrew Geyster, and partner Dustin Taylor.

This project is released as open source, anyone is allowed to download/view/edit code and this is not to be distributed by anyone in such a manner that money is made from it.

Controls:

A - Move Left
D - Move Right
Space - Jump
CTRL - crouch
Shift - Run
Tab - Reset level
K - Debug kill command
